

# TCP/IP Configuration Tool

A simple and efficient tool for rapid configuration of TCP/IP network settings. This tool automates the process of changing IP addresses, subnet masks, and default gateways, eliminating the need for manual navigation through the Control Panel.

 Features
Quick IP Change: Instantly set static IP addresses or switch to DHCP.

DNS Management: Configure preferred and alternate DNS servers.

Status Preview: Display current network adapter parameters in real-time.

Security: Built-in administrator privilege verification before applying changes.

 Installation & Execution
Script Version (Python/PowerShell)
To run the tool as a script:

Download or clone this repository.

Open your terminal with Administrator privileges.

Execute the main script file.

Executable Version (.EXE) 
For users without a pre-installed development environment, a standalone EXE version is available.

No additional libraries or runtimes required.

Simply download TCP_IP_Config.exe from the Releases section.

Note: The file must be Run as Administrator for network setting changes to take effect.

 Requirements
Operating System: Windows 10 / 11.

Permissions: Administrative access (UAC).
